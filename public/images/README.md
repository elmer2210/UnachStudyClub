En esta carpeta deben ir todas las imágenes estáticas, iconos y logos del proyecto (por ejemplo, el logo de la universidad o iconos genéricos).

**Nomenclatura de archivos:** Los archivos deben llamarse usando minúsculas y separando las palabras con guiones medios. 
*Ejemplo:* `logo-unach.png`, `fondo-login.jpg`, `icono-usuario.svg`.
